# Sorry, Anushka — React (JSX)

A heartfelt single-page React app.

## Run

```bash
npm install
npm run dev
```

Then open the URL Vite prints (usually http://localhost:5173).

## Files
- `src/App.jsx` — the page
- `src/App.css` — all styles (cherry blossom theme)
- `src/main.jsx` — React entry
- `src/blossoms.jpg`, `src/hearts.jpg` — images
- `index.html` — HTML shell
